Me especialidad

Soy un desarrollador web especialisodo en la interactividad del ususario UI, creo y doy estilos a paginas webs, con un codigo ordenado y limpio,trabajo principalmente con HTML, CSS y JavaScript, lo que me permite construir sitios web rápidos, responsivos y bien estructurados. Cuido mucho los detalles visuales como colores, tipografía, animaciones y disposición de elementos para lograr un diseño profesional. Mi objetivo es no solo hacer que un sitio se vea bien, sino que también sea cómodo de usar y cumpla su propósito: atraer, comunicar y convertir.

Netflix

Este proyecto es una clonación de la interfaz de Netflix, desarrollada con el objetivo de practicar y mejorar mis habilidades en desarrollo front-end. La aplicación replica el diseño moderno y atractivo de la plataforma, incluyendo secciones como el banner principal

Se implementó utilizando HTML, CSS y JavaScript, enfocándome en la estructura semántica, estilos responsivos. Este proyecto me permitió fortalecer mis conocimientos en diseño UI, maquetación y lógica básica de interacción, acercándome a la construcción de aplicaciones web más completas y profesionales.
